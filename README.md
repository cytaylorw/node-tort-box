# node-tort-box
This is a project made to monitor my tortoise pet indoor enclosure. Integration with Thingsboard platform to visualize the data.

## Hardwares

- Raspberry Pi 3 Model b
- DHT22 temperature and humidity sensor

## Softwares

- Raspbian Lite
- Node.js
- MQTT
